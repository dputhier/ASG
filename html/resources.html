Resources

ASG1 http://pedagogix-tagc.univ-mrs.fr/courses/ASG1/ Web site for this course
R http://www.r-project.org/ A free software environment for statistical computing and graphics
R markdown http://rmarkdown.rstudio.com/ Documentation for R markdown language (used for the practicals)
Bioconductor http://www.bioconductor.org/ A set of R libraries dedicated to statistical analysis of genomics data.
MeV: MultiExperiment Viewer http://www.tm4.org/mev.html A Java application designed to allow the analysis of expression data
Cluster 3.0 http://bonsai.hgc.jp/~mdehoon/software/cluster/software.htm Implements the most commonly used clustering methods for gene expression
java Treeview http://jtreeview.sourceforge.net/ Java-based tool to visualize trees prodced by hierarchical clustering togeter with a heatmap with expression proviles.
Integrative Genome Viewer (IGV) http://www.broadinstitute.org/igv/ A high-performance visualization genome browser.
Tophat http://tophat.cbcb.umd.edu/ A high-performance splice-aware aligner.
Bowtie http://bowtie-bio.sourceforge.net/bowtie2/index.shtml Bowtie: Fast and sensitive read alignment.
FastQC http://www.bioinformatics.babraham.ac.uk/projects/fastqc/ quality control checks on raw sequence data coming from high throughput sequencing pipelines.
MACS (14) http://liulab.dfci.harvard.edu/MACS/ Model-based Analysis for ChIP-Seq. Tha MACS Peak caller.
